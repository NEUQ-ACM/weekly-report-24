#### leetcode 刷题（570. 至少有5名直接下属的经理）
##### 思路
用子查询查担任员工经理>=5的，然后再得到其id
##### 代码
```mysql
select name
from Employee
where id in ( select managerId from Employee group by managerId having count(*)>=5 )
```
